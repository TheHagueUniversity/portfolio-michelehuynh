Deliverable

Design a mobile app that could be used as an educational tool box

Context

Course Project (University, Individual)

Timeline

September 2019 - October 2019 (8 weeks)

Learning Goal & Task

Initial learning goal to takeaway from this project was to translate information into a digital visualisation by creating an appropriate structure and digital design. The task that we were given was designing a mid-fidelity prototype of an educational app that could be used as a learning toolbox for UXD students at my university for the first year course, Design and Creativity.

Inital Idea

Educational resources find their mediums in varying digital forms nowadays. This course project is devoted to the user interface design of a resource library for creative techniques incl. diverging and converging techniques learned during the course. The task was to design a mobile user interface of a mobile app which includes a resource library where newly learned material can be inputed once learned i.e. add a new technique and serve an as educational toolbox. Each technique within the library includes a visual (showing the technique in action), a list of requirements (for best results), and the goal (of the technique and what it can bring to the designer.)

Vision

The main task of an educational application is a convenient and understandable transfer of information. The goal is to create an educational toolbox that is simple, convenient and user-friendly.
I intended for the design to be clean and minimalistic and for the structure to be organised and usable.
